M4 — SurfacePatch + Cinematic Dive (88 unit tests, 9 E2E tests)
  - scripts/fetch-mola-dem.mjs: downloads 256x256 MOLA MEGDR 128ppd crops via HTTP Range
    requests (~5.6 MB per rover vs 129 MB full tile)
  - public/data/dem/perseverance.bin/.json (Jezero Crater, -3323 to -187 m)
  - public/data/dem/curiosity.bin/.json (Gale Crater, 706 to 2731 m)
  - SurfacePatch.tsx: PlaneGeometry displaced by heightfield in tangent frame (5x vert exag),
    only renders when cameraMode === 'surface' + rover selected
  - CameraRig.tsx: two-phase GSAP timeline -- Phase 1 rotate to face rover (1.2s),
    Phase 2 descend to 2km north + 0.3km above rover surface (2.5s) with concurrent lookAt
    tween from planet center to rover surface point; OrbitControls stays disabled in surface
    mode to prevent minDistance snap-back to orbit altitude
  - Scene.tsx: SurfacePatch wired for both rovers, camera near/far widened

COMMANDS
--------
npm run dev        — start dev server at http://localhost:5174
npm test           — run all unit tests (Vitest)
npx playwright test — run E2E tests (headless:false, needs display)
npx tsc --noEmit   — type check
node scripts/bake-traverses.mjs   — regenerate traverse JSON files
node scripts/bake-waypoints.mjs   — regenerate waypoint JSON files
git push           — deploy to GitHub Pages via Actions
